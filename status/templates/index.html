Welcome to the ERGA-Stream Tracking Tool

This site is under active development by the CNAG team (namely, Tyler Alioto). The stable production version will be located at https://genomes.cnag.cat/erga-stream. If you'd like to browse the api or help test the tool, please register an account via the link at the top right and we will manually activate your account and grant limited permissions. If you'd like to help develop, head on over to https://github.com/cnag-aat/ERGA-Status

DISCLAIMER: Things are likely to change and the site may occasionally be down. This is NOT a production site (yet;-)

How to use this tool.

As soon as possible, Sequencing, Assembly, and Annotation leads should register at the link above, and once granted access, enter team member info using the appropriate link below:

  • Edit sequencing teams
  • Edit assembly teams
  • Edit annotation teams

Analogous links exist for other roles, but perhaps the genome team coordinator or sample coordinator can enter this info. The can also be accessed via the genome team link.

Scripts will be provided in the github repo for updating the database with seqeuncing and assembly status. These will take as imput tsv or csv text files with new or updated info.

Reference Genome Assembly Quality Controls

Genome references produced under BGE and ERGA in general should adhere to the EBP standards.
